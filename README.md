# CHOU COLLECTIVE - Website Documentation

## Overview

A modern, responsive website for CHOU COLLECTIVE featuring smooth animations, sticky navigation, and professional design.

## Features

Design & Layout

- Clean, minimalist design with professional aesthetics
- Responsive grid layouts
- Mobile-first approach

Responsive Design

- Desktop: Optimized for screens 1024px+
- Tablet: Adjusted layout for 768px-1024px
- Mobile: Fully functional on 320px-767px

Navigation

- Sticky navbar stays visible while scrolling
- Hover effects on navigation bar, images, buttons
- Dropdown menu with smooth transitions, accordion layout on mobile
- Slide-in side menu with hamburger toggle on mobile
- Auto-highlights current section
- Click navigation smoothly scrolls to sections

Animations

- Content slides up into view
- Left, right, up, down reveal animations
- Images zoom/scale on hover
- Delayed animations for grid items

Accessibility

- Semantic HTML5 tags
- ARIA labels and attributes
- Focus styles for interactive elements
